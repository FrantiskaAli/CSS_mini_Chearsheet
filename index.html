CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favourite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it will automatically select all of the code for you to copy.

Flexbox Row

Use these properties to create flexbox row layoutt.

.row {
display: flex;
flex-direction: row;
flex-wrap: wrap;
}

Flexbox Column

Use these properties to create flexbox row layoutt.

.column {
display: flex;
flex-direction: column
}

CSS Grid Layout

Build column layout using CSS Grid.

.grid {
display: grid;
width: 100%;
grid-template-columns: repeat(12. 1fr);
}

Linear Gradients

This will create background linear gradent from top to bottom.

.linear-gardient-background {
background-image: linear gradient
(180deg, color, color);
}

Box Transition Glow

Use transition and box shadows to glow on hover.

.code-card .card-header {
border-radius: 8px;
transition: all 0.5s ease-in-out;
}
.code-card:hover .code-header:hover {
box-shadow: inset 0px 0px 8px colour,
0 0 15px colour
}

Overlay Card with Title

The position properties navigates margin to raise elements higher then their natural standpoint.

.card-header {
position: relative;
margin-top: -20px;
}
